# 회원제 사이트 만들기

### 사용기술

- html, css, javascript, bootstrap5, php, sql

### 기능

- 회원가입, 수정
- 로그인, 로그아웃
- 글쓰기, 보기, 수정, 조회, 삭제
- 관리자 모드

### 프로그램

- 편집기 (VS CODE)
- xampp
- db관리툴 (sqlyog or heidisql or phpmyadmin)
- 브라우저 (chrome or firefox)
